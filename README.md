# Analytic codes for MR
TSMR_analysis.R------analytic codes for two-sample MR

MVMR_analysis.R------analytic codes for conventional multivarible MR

FLOWMR_analysis.R------analytic codes for FLOW-MR

# Figure codes
BMI_K=4_migraine_direct_plot.R------analytic codes for drawing forest plot which shows the direct effects detected by FLOW-MR

BMI_K=4_migraine_indirect_plot.R------analytic codes for drawing forest plot which shows the indirect effects detected by FLOW-MR

BMI_K=4_migraine_path_plot.R------analytic codes for drawing forest plot which shows the path-wise effects detected by FLOW-MR

BMI_K=4_migraine_total_plot.R------analytic codes for drawing forest plot which shows the total effects detected by FLOW-MR
